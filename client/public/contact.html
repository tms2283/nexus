02 — ApproachThe AI-Native Method
Built with AI. Designed by me.

I build AI systems.
I also build with AI systems.
The difference matters.

The honest truth
Claude wrote a lot of this code
And I designed every system it wrote. I defined the architecture, the data contracts, the edge cases, the user flows. AI compresses weeks into days — it doesn't replace the thinking that makes weeks worth compressing.
What that means in practice
14 agents don't replace judgment
Apex Intelligence runs 14 parallel AI agents to audit websites. I designed each agent's scope, wrote the prompts that govern its behavior, defined what "fabricated data" means and hard-coded rules against it. The agents execute. I architect.
The real advantage
One builder, full-stack output
My background is operations and business ownership, not computer science. AI tools let me operate as a full product team — backend, frontend, design, and QA — while staying focused on what matters: building things people actually need.
The stack I trust
Claude + Cursor + n8n + ComfyUI
Claude for architecture, reasoning, and code review. Cursor for fast implementation. n8n for workflow automation. ComfyUI for AI image pipelines. Ollama for running local models. These aren't crutches — they're the modern power tools.

05 — AboutTim Schmoyer
Background

I'm Tim Schmoyer — a self-directed technical builder based in Fredericksburg, VA. My background is in operations, business ownership, and management across varied industries. I didn't come up through computer science. I came up through getting things done.

That background shapes how I build: I care about outcomes, not elegance for its own sake. A system that runs reliably and solves a real problem beats a beautifully architected one that ships six months late every time.

Today I work across AI-powered SaaS products, automation systems, and creative tooling — often all at once. I strongly prefer autonomous AI execution, delegating broadly to AI agents across Claude Desktop, VS Code, and custom MCP server ecosystems I've built myself.
